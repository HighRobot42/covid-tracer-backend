# covid-tracer-backend
Backend for the [CovidTracer app](https://github.com/RaphaelJ/covid-tracer).

## Usage

    DATABASE_URL=<DATABASE URL>                 \
    SECRET_KEY=<SECRET KEY>                     \
    APP_SETTINGS='config.ProductionConfig'      \
    FLASK_ENV=production                        \
    python3 app.py
